# Primeros pasos con GitHub Copilot

_Comienza a usar GitHub Copilot en menos de una hora._

## Bienvenida

- **¿Para quién es esto?**: Desarrolladores de cualquier nivel de experiencia que buscan acelerar su flujo de trabajo de código.
- **Qué aprenderás**: Las distintas formas de interactuar con Copilot para explicar, escribir, planificar y desarrollar código.
- **Qué construirás**: Guiarás a Copilot para actualizar el sitio web de actividades extracurriculares de Mergington High School.
- **Requisitos previos**:
  - Ejercicio de habilidades: [Introducción a GitHub](https://github.com/skills/introduction-to-github)
  - Familiaridad con [VS Code](https://code.visualstudio.com/)
  - Principios básicos de programación
- **Duración**: Este ejercicio toma menos de una hora en completarse.

En este ejercicio, vas a:

1. Usar un Codespace preconfigurado para ejecutar VS Code en tu navegador.
1. Aprender diferentes opciones de interacción para desarrollar y planificar con GitHub Copilot.
1. Usar Copilot para resumir y revisar tu pull request.

### Cómo iniciar este ejercicio

Simplemente copia el ejercicio a tu cuenta, luego dale a tu Octocat favorito (Mona) **unos 20 segundos** para preparar la primera lección y después **actualiza la página**.
